Renaizant

Renaizant is an ongoing performance management platform that helps employees grow and succeed.

Goals

1

Develop UI/UX, create product concept and vision

2

Use ML and AI for contextual suggestions

3

Add multiple integrations: Jira, SalesForce, GitHub, Slack, Office 365, Plaid, Stripe, etc.

4

Add new features concepts and features to cover a set of standard tools for performance review

Solutions

1

We used a Just-In-Time feedback approach to make performance review a continuous activity

2

Integrated powerful platforms to track the progress of users who even don't use the app

3

Added features for quality, easy, and fun performance review

Outcome

Developed a fully functioning MVP within one year.
